Planar Table 多頁式靜態官網
============================

檔案結構
- index.html：首頁
- product.html：產品特色
- market.html：市場與商模
- team.html：團隊介紹
- roadmap.html：藍圖與策略
- assets/css：樣式檔
- assets/js：互動腳本
- assets/images：由簡報轉出的示意圖
- docs：原始計畫書與簡報

使用方式
1. 直接開啟 index.html 即可瀏覽。
2. 若要部署到 GitHub Pages / Netlify，可整個資料夾直接上傳。
3. 若後續要接表單、會員或 Next.js，可沿用目前資訊架構。
